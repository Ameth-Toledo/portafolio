Despliegue de aplicaciones frontend en AWS EC2 y Nginx 💪🔥

El despliegue de aplicaciones frontend en AWS EC2 con Nginx permite llevar proyectos a producción con alto rendimiento y control total del entorno. Este proceso incluye la configuración del servidor, la instalación de dependencias, la gestión del firewall o puertos, y el uso de Nginx como servidor web para servir aplicaciones de forma eficiente y segura.

Instalación de Nginx

Vamos a instalar y configurar nginx para que nuestra aplicación web Front-End funcione perfectamente.

Seleccionamos nuestra instancia y damos click sobre el boton que dice Conectar, nos llevara a otra vista donde veremos nuetra IP y otras cosas, nos desplazaremos hacia abajo y damos click sobre el boton amarillo que dice conectar y esperamos.

Nos abrira la terminal. Aqui vamos a buscar y actualizar paquetes que nuestro sistema operativo nesesite asi que vamos a ejecutar el comando sudo apt update && sudo apt upgrade -y para buscar y actualizar paquetes.

Despues de que hayamos actualizado nuestros paquetes vamos a instalar Nginx para ellos vamos a ejecutar el comando sudo apt install nginx -y y solo queda esperar a que termine el proceso de instalación.

Ahora regresamos a nuestra app web en Visual Studio Code y tenemos que hacer un build para que el framework nos genere la carpeta dist, el comando para hacer build depende mucho del framework asi que te tocara buscar cual es el comando para hacer build en tu app.

Una ves que tengamos la carpeta dist, vamos a hacer huzo de FileZilla para subir nuestros archivos a EC2 si no lo tienes puedes decargarlos aqui: https://filezilla-project.org/download.php?platform=win64

Una ves de que ya tengamos FileZilla vamos a abrirlo y daremos click sobre el icono de servidores que se encuentra de lado izquierdo, en la imagen se muestra en que opcion se debe de dar click.
