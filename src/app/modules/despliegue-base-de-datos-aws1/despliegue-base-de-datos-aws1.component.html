Despliegue de Base de Datos en AWS EC2 y MySQL 💪🔥

La configuración y gestión de bases de datos en AWS EC2 permite almacenar, consultar y administrar información de manera segura y eficiente en entornos de producción. Este proceso incluye la instalación del motor de base de datos, la creación y configuración de usuarios, la gestión de permisos y respaldos, así como la optimización del rendimiento para garantizar la disponibilidad y la integridad de los datos.

Actualización e instalación de MySQL Server

A continuación, vamos a abrir los puertos necesarios para correr el servidor de MySQL. Abriremos el puerto 3306, que es donde se ejecuta MySQL Aurora. Nos dirigimos al menú inferior y seleccionamos la opción de Seguridad. Luego, desplazamos un poco hacia abajo y vamos a Grupos de seguridad. Damos clic en el enlace y se abrirá otra ventana, donde haremos clic en Editar reglas de entrada.

Después vamos a dar clic en Agregar regla y en tipo vamos a seleccionar MySQL Aurora en origen vamos a seleccionar 0.0.0.0/0 para que se accesible a todas las conexiones remotas. Damos clic en Guardar reglas y nos llevara a la interfaz de donde estan todos los puertos abiertos y ya veremos el puerto 3306 abierto, ahora si ya podemos actualizar e instalar MySQL.
