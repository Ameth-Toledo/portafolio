Despliegue de Base de Datos en AWS EC2 y MySQL 💪🔥

La configuración y gestión de bases de datos en AWS EC2 permite almacenar, consultar y administrar información de manera segura y eficiente en entornos de producción. Este proceso incluye la instalación del motor de base de datos, la creación y configuración de usuarios, la gestión de permisos y respaldos, así como la optimización del rendimiento para garantizar la disponibilidad y la integridad de los datos.

Creación de instancia EC2 con IP elástica

El primer paso es dirigirnos al laboratorio de AWS y le damos en iniciar laboratorio de aprendizaje, nos abrirá otra ventana en el navegador, en donde nos saldrán varias opciones como: Start Lab, End Lab etc. Daremos clic sobre Start Lab y esperamos a que el estado de AWS se cambie a verde.

Después nos mostrará otra vista en donde tendremos un buscador, ahí vamos a buscar EC2 y damos clic sobre la primera opción, en mi caso aparece como visitados recientemente, damos clic y nos mandará a la sección de instancias, en mi caso tengo creada una llamada RabbitMQ, a ustedes les aparecerá sin instancias.

Ahora haremos clic sobre el botón amarillo que se encuentra del lado derecho que dice lanzar instancia, nos mandará a la sección de configuración, nos va a pedir un nombre de instancia, en mi caso le pondré Base de Datos aquí el nombre no importa.

Ahora vamos a seleccionar el sistema operativo de nuestra máquina, aquí es importante seleccionar Ubuntu, ya que si seleccionan otro sistema operativo el despliegue no va a funcionar correctamente, luego de haber seleccionado el sistema operativo nos desplazamos y encontraremos la sección de par de claves, vamos a dar clic sobre continuar sin par de claves.

Una vez que hayamos hecho todo lo anterior nos mostrará una ventana indicando que el lanzamiento de la instancia fue correcto, damos clic sobre el botón amarillo que dice ver todas las instancias, nos llevará a otra vista ahí vamos a ver la instancia que acabamos de crear, debemos esperar hasta que el estado de comprobación cambie a 3/3 comprobaciones y listo ya quedaría.
