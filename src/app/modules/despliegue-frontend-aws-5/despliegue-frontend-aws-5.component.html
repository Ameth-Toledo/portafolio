Despliegue de aplicaciones frontend en AWS EC2 y Nginx 💪🔥

El despliegue de aplicaciones frontend en AWS EC2 con Nginx permite llevar proyectos a producción con alto rendimiento y control total del entorno. Este proceso incluye la configuración del servidor, la instalación de dependencias, la gestión del firewall o puertos, y el uso de Nginx como servidor web para servir aplicaciones de forma eficiente y segura.

Instalación de Nginx

Vamos a instalar y configurar nginx para que nuestra aplicación web Front-End funcione perfectamente.

Seleccionamos nuestra instancia y damos click sobre el boton que dice Conectar, nos llevara a otra vista donde veremos nuetra IP y otras cosas, nos desplazaremos hacia abajo y damos click sobre el boton amarillo que dice conectar y esperamos.

Nos abrira la terminal

En Origen vamos a seleccionar Anywhere y quedaria configurada el primer protocolo HTTP.
Ahora vamos a configurar el protocolo https, este protocolo corre en el puerto numero 443, hacemos el mismo proceso de configuración.

Una ves de que ya hayamos abierto los puertos, daremos click en Guardar reglas.
Nos mostrara el mensaje de exito una ves que hayamos guardado los cambios y ahora en la lista ya apareceran los puertos que configuramos.

Ve a la siguiente pagina para continuar con el despliegue.
