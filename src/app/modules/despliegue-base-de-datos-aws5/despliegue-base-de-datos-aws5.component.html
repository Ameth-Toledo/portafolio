Despliegue de Base de Datos en AWS EC2 y MySQL 💪🔥

La configuración y gestión de bases de datos en AWS EC2 permite almacenar, consultar y administrar información de manera segura y eficiente en entornos de producción. Este proceso incluye la instalación del motor de base de datos, la creación y configuración de usuarios, la gestión de permisos y respaldos, así como la optimización del rendimiento para garantizar la disponibilidad y la integridad de los datos.

Verificación de la Configuración de MySQL

A continuación, vamos a verificar la configuración. Para ello, copiamos nuestra IP y abrimos Workbench para crear una nueva conexión y comprobar que el acceso remoto funcione correctamente.

Ahora vamos a dar clic en el ícono de +. Nos pedirá un nombre; aquí puede ser cualquier nombre, ya que es solo el nombre de la conexión. En el campo Hostname pondremos la IP que copiamos de la instancia, y en Puerto dejamos el valor 3306. En Username ingresamos el usuario que creamos; en mi caso es ameth. Una vez llenada la información, damos clic en OK. La conexión creada aparecerá en la lista de conexiones.

Ahora damos doble clic sobre la conexión creada y nos aparecerá una ventana. Allí vamos a ingresar nuestra contraseña para el usuario creado. Una vez que ingresamos la contraseña, damos clic en OK y esperamos a que la conexión se establezca.
