Despliegue de Base de Datos en AWS EC2 y MySQL 💪🔥

La configuración y gestión de bases de datos en AWS EC2 permite almacenar, consultar y administrar información de manera segura y eficiente en entornos de producción. Este proceso incluye la instalación del motor de base de datos, la creación y configuración de usuarios, la gestión de permisos y respaldos, así como la optimización del rendimiento para garantizar la disponibilidad y la integridad de los datos.

Configuración de IP Elastica

A continuación, vamos a crear nuestra IP elástica para luego conectarnos a Workbench. En el menú del lado izquierdo nos dirigimos a la sección de Direcciones IP elásticas. Ahí encontraremos todas las IP elásticas que hayamos creado. En mi caso, aparece una llamada BD Ejemplo; a ustedes les aparecerá vacío. Vamos a crear una nueva dirección IP haciendo clic en el botón Asignar dirección IP elástica.

Se abrirá otra ventana; no modificamos nada, solo daremos clic en el botón Asignar. Luego nos regresará a la vista donde se encuentran las IP elásticas. Ahora vamos a asignarle un nombre para identificar la IP. Una vez asignado el nombre, haremos clic en el botón verde que dice Asociar esta dirección IP elástica.

Una vez que hagamos clic en el botón Asociar la dirección IP elástica, se abrirá otra ventana. En la parte que dice Instancia, desplegaremos el menú y buscaremos nuestra instancia donde instalamos MySQL.

Todo lo demás lo dejamos igual. Ahora sí, simplemente hacemos clic en el botón amarillo que dice Asociado. Luego regresamos a la sección de Instancias. y la ip ahora ya cambio.
